ไฟล์ .sql สำหรับสร้าง Database
sql_structure.sql เก็บข้อมูลการสร้าง Database เฉพาะโครงสร้าง Database
any_i.sql เก็บทั้งโครงสร้าง และข้อมูล ตามใน Demo ของทาง Neumerlin
**ระวังการใช้ไฟล์สำหรับสร้าง ฐานข้อมูล ไฟล์จะทำการ Drop Table และ ข้อมูลทั้งหมดใน Database ก่อนที่จะไส่ข้อมูลไหม่ ข้อมูลเก่าใน Database จะถูกลบออกทั้งหมด

Database Connection สามารถตั้งค่าได้ที่ไฟล์ ./model/db.php

Content Web ภาษาไทย และภาษาอังกฤษ แก้ใขได้ที่ไฟล์ ./model/lang.php
